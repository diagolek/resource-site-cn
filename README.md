# resource-site-cn

## 项目简介

本项目用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存储，不依赖特定后端服务，可直接通过浏览器打开查看。仓库内容主要面向中文用户，涵盖各类信息展示、工具页面或资源汇总页面。

## 目录说明

```
resource-site-cn/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页或导航页面（可选）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件代表一个独立的页面，文件名通常反映其内容主题。
- **assets/**：页面所需的外部资源文件，按类型或功能组织子目录。
- **index.html**：可选入口页面，用于汇总或引导访问者浏览各个页面。

## 页面归档说明

所有页面均为静态 HTML，无动态数据请求，可离线或在线直接查看。归档内容遵循以下原则：

- 页面内容保持中立、客观，不包含特定商业或政治立场。
- 每个页面独立完整，不依赖外部不可控资源（如第三方 API）。
- 如需引用外部资源（如公共 CDN 库），会在页面内明确标注来源。

## 维护说明

- 本仓库接受新增或更新 HTML 页面的提交，但需确保内容符合上述归档原则。
- 欢迎通过 Issue 或 Pull Request 提出建议或贡献页面。
- 仓库维护者会定期检查页面可用性，并处理明显的链接或显示问题。
- 如有页面不再适用或需要移除，可通过提交 PR 删除对应文件并说明原因。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/resource-site-cn.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可将整个目录部署到静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
